用户 API
获得用户列表 保存用户 删除指定编号的用户
目录

获得用户列表

请求URL

/api/user/list GET

请求参数 application/x-www-form-urlencoded

参数名 类型 必须 描述
nickname string 否 昵称，模糊匹配

返回结果

[{
	"id":"int //用户编号",
	"nickname":"string //昵称",
	"age":"int //年龄"
}]
